# AviutlでWindowを生成するPlugin

![img](https://github.com/YucriNatsure/Create-new-window-Aviutl/blob/main/script/img/Example.PNG "img")

このPluginはAviutl上で新しいWindowを生成するものです。

自分はこれで自作アプリを外部実行するPluginを作りました()
